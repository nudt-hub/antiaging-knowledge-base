## 📖 工具使用指南

### 如何选择工具？

| 需求 | 推荐工具 | 理由 |
|------|---------|------|
| **评估整体衰老速度** | PhenoAge | 综合血液指标，准确性高 |
| **监测干预效果** | PhenoAge | 可重复检测，成本较低 |
| **研究特定基因** | HCATA | 单细胞分辨率，组织特异性 |
| **探索衰老机制** | HCATA | 通路富集分析，深入机制 |
| **查询衰老基因** | 衰老基因检索 | GenAge 数据库，300+ 基因 |
| **探索长寿物种** | 衰老基因检索 | AnAge 数据库，4000+ 物种 |
| **发现干预措施** | ClockBase Agent | 500+ 有效干预，AI 驱动 |
| **追踪最新研究** | ClockBase Agent | 每周自动监控 |

### 联合使用建议

**最佳实践**：

1. **基线评估**：使用 PhenoAge 计算生物学年龄
2. **机制探索**：使用 HCATA 查询衰老相关基因
3. **基因研究**：使用衰老基因检索深入探索
4. **干预发现**：使用 ClockBase Agent 发现有效干预
5. **定期监测**：每 3-6 个月复查 PhenoAge
